Referensi praktis agar input kalkulator konsisten dan mudah dipertanggungjawabkan.

Kalkulator Panduan

Asumsi Dasar dan Rentang Referensi

Gunakan angka berikut sebagai baseline awal sebelum kalibrasi lapangan.

Parameter ini menjelaskan sumber energi utama perangkat.

  • Baterai Lithium umumnya memiliki tegangan nominal 3.7V.
  • Nilai ini aman digunakan sebagai baseline perhitungan awal.
  • Jika memakai konfigurasi lain, sesuaikan sesuai datasheet baterai.

Parameter ini menunjukkan tegangan operasi logika perangkat.

  • ESP32 dan SX1276 bekerja pada domain 3.3V.
  • Perhitungan energi transmisi dan standby menggunakan nilai ini.
  • Pastikan regulator dan catu daya stabil agar estimasi mendekati kondisi nyata.

Arus standby adalah arus saat node aktif tetapi tidak sedang transmit.

  • Rentang idle yang sering dijumpai: 40-50 mA.
  • Saat proses sensor/aplikasi aktif: sekitar 60-90 mA.
  • Gunakan nilai rata-rata pengukuran lapangan agar hasil lebih presisi.

Arus transmisi dipakai saat modul LoRa sedang mengirim paket.

  • Pada 20 dBm, SX1276 umumnya sekitar 120 mA.
  • Variasi implementasi lapangan: sekitar 105-130 mA.
  • Perbedaan dipengaruhi modul, antena, dan kualitas catu daya.

Airtime menentukan durasi radio aktif setiap transmisi.

  • SF7: 62 ms
  • SF8: 113 ms
  • SF9: 205 ms
  • SF10: 371 ms
  • SF11: 741 ms
  • SF12: 1319 ms

Semakin tinggi SF, jangkauan meningkat tetapi konsumsi energi juga naik.

Interval kirim menentukan seberapa sering perangkat mengirim data.

  • Ikuti nilai TX_INTERVAL pada firmware agar konsisten.
  • Interval lebih rapat meningkatkan energi per jam.
  • Interval lebih longgar memperpanjang umur baterai.
